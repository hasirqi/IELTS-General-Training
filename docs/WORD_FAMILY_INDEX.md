# 20K 词族测量索引

这个索引服务于词汇量测量、CAT 选题和文本覆盖分析，不是要求用户学习
20,000 个单词，也不等于现有 4,133 条教学词库。

## 状态门禁

词族与锚点按以下状态推进：

1. `frequency-seeded`：仅由词频和保守形态规则生成，不参与计分。
2. `expert-reviewed`：人工确认词族边界、词性、核心义和层级，仍不参与计分。
3. `item-authored`：四选一题目、唯一答案和三个有效干扰项完成，可进入实验计分。
4. `pilot-active`：经过内容门禁和算法模拟的实验锚点。
5. `calibrated`：未来有足够自愿样本后完成统计标定。

正式运行只允许 `item-authored`、`pilot-active` 和 `calibrated` 进入计分。
这条规则由运行时和自动测试同时执行，不能由 UI 绕过。

## 词族处理规则

- 屈折变化和一组明确的不规则变化归入候选词族。
- 透明派生只做候选归并；进入锚点时必须重新人工确认，防止错误合并。
- 多词词块映射到组成词族，不额外膨胀“词族量”。
- 专有名词、数字、日期和缩写不计入 20K 主尺度，单独标记和分析。
- 同形异义词在频率底座中共用词形，但锚点按明确词性和义项出题。
- 每次构建都保存版本；旧测试结果保留原版本，不静默改写。

## 数据来源

频率种子来自 `wordfreq 3.1.1`。项目保存派生排名、Zipf 频率和来源信息，
并遵守其 Apache-2.0 代码许可及 CC BY-SA 4.0 数据许可和上游署名要求。
详细署名见仓库根目录 `THIRD_PARTY_NOTICES.md`。

本索引的 20,000 行仍是候选骨架，不宣称等同于 Nation/BNC/COCA 的人工词族表。
任何候选只有通过人工锚点审核后才会影响用户得分。

## M1 构建结果（2026-07-24）

- 20,000 个唯一候选词族，频率排名 1–20,000 连续。
- 4,133 条教学词全部有明确处理结果。
- 直接命中 3,187；词族成员命中 121；多词词块组件映射 121；未匹配 704。
- 未匹配不等于删除，表示需要在后续人工审核中决定是否属于低频词、专名、拼写变体或超出当前 20K 上限。
- 全部候选保持 `frequency-seeded`，不会影响用户分数。
