# 破壁 IELTS 6

面向英语词汇基础薄弱学习者的 IELTS General Training 本地优先学习 PWA。中文负责讲清概念，学习任务强调主动回忆、延迟复习、真实语境和可验证的进步。

## 当前已实现

- 1,000 条词与词块全部进入每日学习、复习调度、错词重现和延迟回忆。
- 582 条已完成“词形、词义、用法”人工审核；其余 418 条明确标记为“当前只练词形与核心义”，不使用模板例句冒充内容。
- 128 节 General Training 课程：听力、阅读、写作、口语各 32 节。
- 256 道可确定评分的听力与阅读题；问题使用英语，解释提供中文证据定位。
- 32 段固定英音听力；学习模式可回放，模考模式只能播放一次、不能拖动，交卷后才显示原文。
- 36 个句型挑战、36 个口语练习和 36 周学习路线。
- 写作与口语可复制结构化提示词并打开 ChatGPT 网页；反馈需要由学习者粘贴回产品后才会本地保存，产品不会声称能读取网页会话。
- IndexedDB 本地保存学习状态；没有 OpenAI API Key 也能完成核心学习闭环。
- 键盘焦点、发音按钮名称、字号与移动端触控布局已覆盖。

## 内容质量原则

课程题目必须有唯一可验证答案；主观任务必须给出明确的 6 分检查标准；未完成审核的词条不会获得伪造例句。内容测试会检查重复项、占位文本、答案有效性、课程结构和音频文件。

## 本地开发

```bash
pnpm install
pnpm test
pnpm dev
pnpm build
```

推送到 `main` 后，GitHub Actions 会测试、构建并发布到 GitHub Pages。
