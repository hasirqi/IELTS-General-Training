# 破壁 IELTS 6

面向英语词汇和句型基础薄弱成年学习者的 IELTS General Training 本地优先学习 PWA。中文负责讲清概念，学习任务强调主动回忆、延迟复习、真实语境和可验证进步。

## 任何终端从这里开始

1. 阅读 [项目介绍与共同决策](PROJECT_OVERVIEW.md)。
2. 阅读 [详细项目计划书](PROJECT_PLAN.md)。
3. 执行代理继续阅读 [AGENTS.md](AGENTS.md)。
4. 以 GitHub main 为跨电脑唯一开发基准。

## 当前已实现

- 4,133 条词与词块全部完成词形、聚焦义项、独立真实语境、常用搭配和场景标签审核，并进入学习、复习和错词系统。
- 512 节 General Training 课程：听力、阅读、写作、口语各 128 节。
- 1,024 道可确定评分的听力与阅读题。
- 128 段固定英式神经语音 MP3。
- 192 个句型挑战、192 个口语任务和 36 周学习路线。
- 写作与口语采用复制提示词、打开 ChatGPT 网页、手动粘贴反馈的本地闭环。
- IndexedDB 本地保存学习状态；没有 OpenAI API Key 也能完成核心学习。
- 20,000 词族后台测量索引与首批 150 个双语义项锚点已经完成；测试中心已上线快速定位、英文语境词汇 CAT 和中文释义快测。
- 当前全量自动测试为 211 项。

## 下一阶段

雅思核心词扩展 16 批已经结束。下一阶段建设：

- 将 150 锚点实验版扩展至 600 锚点可用版，再扩展到 1,000 锚点完整版。
- 720 道等级题、60 篇功能文本和 50 篇连续文章。
- 内部文本难度引擎与阅读 CAT。
- 教学词掌握、接受性词族、内部模拟阅读值和 GT Reading 准备度三轴结果。

内部模拟阅读值不是 MetaMetrics 官方认证，词汇量也不会直接换算 IELTS 总分。

## 内容质量原则

课程题目必须有唯一可验证答案；主观任务必须给出明确的 6 分检查标准；任何词条都不能使用模板例句或占位语境。内容测试检查重复项、占位文本、答案有效性、课程结构和音频文件。
